# HowTo export animated gif from multilayer image in Gimp

1. Open or create project in Gimp
2. Add layers
3. Rename layers with (100ms) as speed of animation in suffix name
4. Image -> Mode -> RGB
5. Filters -> Animation -> Unoptimized
6. File -> Export -> By extension -> GIF
7. In pop-up window of export:
   - Checkbox: As animation
   - Frame disposal when unspecified: One frame per layer (change)
   - Export
